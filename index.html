Tree Component for AngularJS

Tree Control is an AngularJS component that can add/remove siblings and child. Zero dependency on jQuery. Recursive implementation.

View on GitHub Download .zip Download .tar.gz

I have tried number of tree control but none is available out in open source community which can assist you to add siblings too. Since I want to control number of child's that can be added to single node. So I came up with my own tree control implementation.

Features

  • Data binding for the tree model
  • Data binding for the selected node in the tree
  • Adding siblings
  • Customize number of child's to be added on a single node(Default: 6)
  • Remove node
  • Recursive controller and HTML code
  • Can be customized with CSS
